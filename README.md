### Movie Recommender API

This project is a RESTful API built with FastAPI that takes a list of movies a user has liked and uses the Gemini AI model to generate personalized movie recommendations. Users can create a profile, save movies they enjoy, and call a single endpoint to get five AI-powered suggestions tailored to their taste. All data is stored in a lightweight SQLite database, and the API is deployed live on Replit with a public URL. It directly fulfills the MLH Cloud Challenge requirement of building and deploying a RESTful API to a cloud service.
Available at :-https://movie-recommender-api-bpdn.onrender.com/
